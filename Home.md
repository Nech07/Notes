# Research LLM Wiki

This vault is organized around a simple contract:

- `raw/` holds immutable source material.
- `wiki/` holds LLM-maintained notes, syntheses, and outputs.
- `AGENTS.md` defines how the agent should ingest, query, and lint the vault.
